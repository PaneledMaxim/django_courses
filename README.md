# University — учебный Django‑проект

Проект состоит из двух приложений:

- `catalog` — статический каталог курсов и авторов.
- `schedule` — управление преподавателями, курсами и студентами.

## Функциональность

- выбор приложения на главной странице
- каталог курсов и авторов (список и карточки)
- список преподавателей, создание/редактирование/удаление
- карточка преподавателя и редактирование доп. информации
- курсы с фильтром по преподавателю, создание/редактирование/удаление
- список студентов, создание/редактирование/удаление
- запись студентов на курсы и снятие с курса

## Стек

- Python
- Django 4.2.28
- SQLite
- HTML templates

## Быстрый старт

1. Перейдите в папку проекта:

```powershell
cd university
```

2. Создайте и активируйте виртуальное окружение:

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
```

3. Установите зависимости:

```powershell
pip install Django==4.2.28
```

4. Примените миграции:

```powershell
python manage.py migrate
```

5. Запустите сервер разработки:

```powershell
python manage.py runserver
```

Откройте в браузере `http://127.0.0.1:8000/`.

## Важно для разработки

- В `university/settings.py` стоит `DEBUG = False`, для локальной разработки обычно удобно включить `True`.
- База данных по умолчанию — `db.sqlite3`.
